# TC1031.3.3

## add()
Agrega un dato al arbol.\
Complejidad O(log n)

## remove()
Elimina un valor del arbol y lo imprime.\
Complejidad O(log n)

## find()
Busca e imprime un valor del arbol.\
Complejidad O(log n)

## inorder()
Imprime todos los valores del arbol en un orden específico recursivamente.\
Complejidad O(log n)

## size()
Regresa el tamaño del arbol.\
Complejidad O(1)
